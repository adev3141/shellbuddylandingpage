ShellBuddy

Streamline your workflow with easy access to Git commands and more.

Installation

npm install -g shellbuddy@beta

Usage

Start using ShellBuddy with simple, natural language queries:

buddy how to undo the last commit

Output: git revert HEAD

Features

Git Operations

  • Comprehensive Git Command Library: Access a wide range of Git commands for common and advanced operations.
  • In-Depth Repository Status: Get a detailed overview of your repository, including branch status, recent commits, and more.
  • Commit Message and Tagging: Easily commit changes with a message and optional tag, with automated checks for changes before committing.
  • Detailed Commit History: View the details of the last commits, including the files changed in each commit.

Environment Management

  • Virtual Environment Management: Easily create, manage, and close Python virtual environments.
  • Automated Virtual Environment Checks: Automatically check if a Python virtual environment exists and create one if necessary.
  • Environment and Dependency Management: Read environment and dependency information to provide commands for running the application efficiently.

System Utilities

  • System Statistics: Quickly check system performance metrics with integrated tools like htop.

AI Interaction

  • Intuitive NLP Interface: Ask questions in plain English and get precise Git commands.
  • Real-Time Content Generation with AI: Interact with Llama3 for generating responses and content based on user prompts.
  • Handle Unknown Commands with AI Assistance: Automatically generate responses for unrecognized commands using the Gemini AI model.

Custom Commands

  • Custom Command Execution: Define and run custom commands to streamline your workflow.
  • Interactive Git Command Menu: Access an interactive menu for selecting and executing common Git commands.
  • Extendable and Adaptable: Add new features and custom commands to suit your unique needs.
